Explanations

Table entries explained
The list type determines whether the videos of this channel are filtered with a whitelist or a blacklist. The values are separated by commas, so the filter filters using the values between two commas.
(Note: Do not filter by anything containing a comma)
Search
You can search for values and channel names.
(Note: The search does not work with the visible commas in the values column)
List types explained
Whitelist means everything but the values is blocked. Blacklist means everything that contains one of the values is blocked.
Storage types explained
Local storage means your settings are only available on this computer. Sync storage means your settings are synced via your firefox/chrome account.
(Note: We do NOT store any data, it is ALL managed by firefox/chrome)
